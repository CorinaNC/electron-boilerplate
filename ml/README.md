## 1. Prepare your environment:
- We are using python 3.11 virtual environment.
- To learn about your python versions:
```
# Run this to know your current version
python3 --version

# Run this to know your installed versions
py --list
```
- Installing your python 3.11 through package manager or the official website: https://www.python.org/downloads/release/python-3110/.
- Create a virtual environment with python 3.11:
```
py -3.11 -m venv .venv
```
- Activate your virtual environment:
```
.\.venv\Scripts\activate
```
- Deactivate your virtual environment:
```
deactivate
```

## 2. Install dependencies:
> Note: please activate your environment first
```
pip3 install -r requirements.txt
```

## 3. Run:
> Note: please activate your environment first
```
python3 my_file.py
```
